# Personal-Portfolio
This is a personal portfolio website developed to showcase my academic background, technical skills, and software development projects. The website serves as a central platform to present my work, interests, and professional profile as a Software Engineering student.
The design is clean, modern, and easy to navigate, making it suitable for academic evaluation as well as professional presentation.

## Features
- About Me section introducing academic background and interests
- Projects section with brief descriptions and GitHub links
- Skills and tools section highlighting technical expertise
- Contact section with email and professional profile links
- Responsive and user-friendly design
- Modern styling with gradients, cards, and background images

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Run
1. Clone the repository:
2. Open the project folder
3. Open `Portfolio.html` in any modern web browser
